# 想法记录本

一个简单易用的想法记录Web应用，让你随时记录灵感，随时查看想法。

## 功能特性

- 📝 **快速记录**：随时记录你的想法和灵感
- 🔍 **智能搜索**：通过标题、内容或标签搜索想法
- 🏷️ **标签分类**：为想法添加标签，方便分类管理
- 📱 **响应式设计**：在手机、平板和电脑上都能完美使用
- 💾 **本地存储**：使用浏览器localStorage保存数据，无需服务器
- 🎨 **美观界面**：现代化的UI设计，舒适的视觉体验

## 使用方法

1. 在"添加新想法"区域填写：
   - 标题（可选）
   - 想法内容（必填）
   - 标签（用逗号分隔，可选）

2. 点击"保存想法"按钮或按 Ctrl+Enter 保存

3. 在"我的想法记录"区域查看所有记录

4. 使用搜索框或标签过滤器快速找到特定想法

5. 点击删除按钮移除不需要的想法

## 技术栈

- HTML5
- CSS3 (Flexbox, Grid, 渐变, 动画)
- JavaScript (ES6+)
- Font Awesome 图标
- localStorage 数据存储

## 部署

- GitHub Pages
- Netlify
- Vercel
